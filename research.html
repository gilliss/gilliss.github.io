  • Improving Read Performance in Write-Optimized Key Value Stores (Jan '15 - Current)
    Guide: Prof. S. Sudarshan, IIT Bombay
    Indexing techniques optimized for a higher write throughput such as LSM Trees generally compromise on read performance. We are working on designing and implementing an indexing technique optimized for both read and write latencies on HBase, an open source implementation of Google's Big Table

  • Automated Linguistic Personalization of Ad Messages (May '14 - July '14)
    Guide: Dr. Rishiraj Saha Roy, Max Planck Institute for Informatics
    Designed and developed a novel method of personalizing ad messages based on the linguisticstyle of a target segment identified by age, gender, occupation, location. Extracted appropriate modifiers from advertisements and inserted them in the text to decrease cross entropy with language model of target segment

  • Synthesis Modulo Bisimulation of Distributed Automaton Systems (Dec '13 - Feb '14)
    Guide: Prof. Madhavan Mukund, Chennai Mathematical Institute
    We are working on the open problem of synthesizing distributed transition systems from global specifications with bisimulation as equivalence criterion. We have gathered evidence that the collapse in state space when one obtains the bisimulation quotient of the distributed transition system can be quantified

  • L2 Readability Analysis (Aug '14 - Dec '14)
    Guide: Prof. Pushpak Bhattacharya, IIT Bombay
    Performed a complete study of existing techniques in estimating readability of a text, in particular for second language learners. Created benchmark metric for readability for Indian L2 learners based on NCERT textbooks for grade levels from 4 to 12

Key Course Projects

  • Cost Metric for Query Optimization (Oct-Dec 2014)
    Guide: Prof. S. Sudarshan, IIT Bombay
    Enhanced size estimation using additional histogram information and thereby the efficiency of the cost metric in the PyroJ query optimizer to achieve resourceful plans for execution

  • Virutal Memory Management System (Mar-Apr 2014)
    Guide: Prof. Dhananjay Damdhere, IIT Bombay
    Developed multi-level page table virtual memory management for an UNIX-based OS. Designed and implemented efficient data structures and algorithms for handling process memory allocation, swap space management and page replacement strategies

  • Parallel Processing in Multi-core Architectures (Oct-Dec 2013)
    Guide: Prof. Bernard Menezes, IIT Bombay
    Analyzed performance gains in multi core architecture for representative problems such as Merge Sort, Matrix Multiplication, Monte-Carlo Simulation and the Bellman Ford algorithm. Estimated the degree of parallelism through techniques such as the Karp Flatt metric

  • Control Flow Graph Language Processor (Feb-Apr 2014)
    Guide: Prof. Udhay Khedkar, IIT Bombay
    Developed a language processor for control flow graph dumps generated by gcc. Implemented a compiler as well as an interpreter for the language

  • Inferring Power Relations from Social Interactions (Feb-Apr 2014)
    Guide: Prof. Pushpak Bhattacharya, IIT Bombay
    Studied how conversational behaviour in social interactions can reveal power relations among participants. Explored efficacy of various learning models in characterizing linguistic styles impacted by power differentials
